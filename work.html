    Ruby on Trails is a hiking app for users to explore and uplaod their favorite hikes. This Full Stack app was my final project for the Web Development Immersive program at General Assembly. Built with Ruby on Rails, it uses Devise for user authentication, the Ospry API for image upload, Google Maps API and geocoding to place location markers on the map, and Disqus for users to leave comments.
